# Go-lang gRPC

Repositório para estudar a aplicação do gRPC

## Como rodar
- Instalar o [protoc](https://grpc.io/docs/protoc-installation/).
- Instalar os [plugins do protoc para Go-lang e gRPC](https://grpc.io/docs/languages/go/quickstart/).
